Readers' Haven – Website Planning Document

Site Name

Readers' Haven – This name suggests a peaceful and welcoming place for book lovers and new readers alike. It’s about creating a personal and relaxing space to find new reads and improve reading habits.

Optional domain availability: readershaven.net

Site Purpose

This website provides book recommendations by genre (mystery, fantasy, nonfiction, etc.), tips for building lasting reading habits, short personal reviews, and curated lists. It also includes a JavaScript-powered "Book of the Week" feature and a form where visitors can share their own recommendations. The site will be responsive and clean to create a pleasant reading experience for all visitors.

Scenarios

  • What book should I start with if I’m new to reading for fun?
  • Can I submit a book I loved for others to discover?

Color Schema

The website will use the following color scheme for visual consistency:

Primary Accent
  • Primary Color (#4A3F35): Used for titles, main navigation, and borders.
  • Accent Color (#88AB8E): Used for buttons, highlights, and section dividers.

Typography

  • Playfair Display (serif): Used for titles and headings to give a refined, literary feel.
  • Open Sans (sans-serif): Used for all body text to ensure readability on screens.
